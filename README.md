## 中文
这是一个可以将所有数据库配置为简单恢复的快速脚本，不仅仅可用于用户数据库，还包括系统数据库。


## English
Here is a quick script to configure ALL your databases to Simple Recovery not just your User Databases, but System Databases too.

---
## Logic
```SQL

use master;
set nocount on
declare @change_all_databases_to_simple_recovery varchar(max) set @change_all_databases_to_simple_recovery = ”
select @change_all_databases_to_simple_recovery = @change_all_databases_to_simple_recovery + ‘alter database [‘ + name + ‘] set recovery simple;’ + char(10) from sys.databases where name not in (‘tempdb’) order by name asc
exec (@change_all_databases_to_simple_recovery)

```
